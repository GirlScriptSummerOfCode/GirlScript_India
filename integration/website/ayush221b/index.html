  • GSSOC - 2018

    GirlScript took the initiative to commence a GirlScript Summer Of Code -a 3 month long project under GirlScript India. Over 1000 students developers and 80 plus mentors registered for the event. It not only aims to incorporate the pure accolades but also individuals new to open source. Be it a rookie or a veteran developer GirlScript Summer Of Code offers something for everyone.
